## Karlılık KPI'ları

### EBIT (Faiz ve Vergi Öncesi Kar)

**Tanım:** Earnings Before Interest and Taxes. Şirketin faiz ve vergi giderleri düşülmeden önceki faaliyet karıdır.

**Formül:**
```
EBIT = Satış Geliri - Satılan Malın Maliyeti - Faaliyet Giderleri
```

veya

```
EBIT = Net Kar + Faiz Gideri + Vergi Gideri
```

**Örnek Hesaplama:**
- Satış Geliri: 10.000.000 TL
- Satılan Malın Maliyeti: 5.000.000 TL
- Faaliyet Giderleri: 2.500.000 TL
- **EBIT = 10.000.000 - 5.000.000 - 2.500.000 = 2.500.000 TL**

**Kullanım Alanı:** Operasyonel verimliliği ölçer, finansman yapısından bağımsız performans gösterir.

---

### EBITDA (Faiz, Vergi, Amortisman Öncesi Kar)

**Tanım:** Earnings Before Interest, Taxes, Depreciation, and Amortization. Nakit yaratma kapasitesini gösterir.

**Formül:**
```
EBITDA = EBIT + Amortisman + İtfa Payları
```

**Örnek Hesaplama:**
- EBIT: 2.500.000 TL
- Amortisman: 300.000 TL
- İtfa Payları: 50.000 TL
- **EBITDA = 2.500.000 + 300.000 + 50.000 = 2.850.000 TL**

---

### Gross Margin % (Brüt Kar Marjı)

**Tanım:** Satışların ne kadarının brüt kar olarak kaldığını gösterir.

**Formül:**
```
Gross Margin % = ((Satış Geliri - SMM) / Satış Geliri) × 100
```

**Örnek Hesaplama:**
- Satış Geliri: 1.000.000 TL
- SMM (Satılan Malın Maliyeti): 400.000 TL
- **Gross Margin % = ((1.000.000 - 400.000) / 1.000.000) × 100 = %60**

**Hedef:** Perakende sektöründe genellikle %50-70 arası hedeflenir.

---

### Net Profit Margin (Net Kar Marjı)

**Tanım:** Tüm giderler düşüldükten sonra satışların ne kadarının net kar olduğunu gösterir.

**Formül:**
```
Net Profit Margin % = (Net Kar / Satış Geliri) × 100
```

**Örnek Hesaplama:**
- Net Kar: 500.000 TL
- Satış Geliri: 5.000.000 TL
- **Net Profit Margin = (500.000 / 5.000.000) × 100 = %10**

---

### Markup (Kar Çarpanı)

**Tanım:** Ürünün satış fiyatının maliyet fiyatına oranıdır.

**Formül:**
```
Markup = Peşin Satış Fiyatı (PSF) / Satın Alma Maliyeti (SMM)
```

veya yüzde olarak:

```
Markup % = ((PSF - SMM) / SMM) × 100
```

**Örnek Hesaplama:**
- SMM: 100 TL
- PSF: 350 TL
- **Markup = 350 / 100 = 3.5×**
- **Markup % = ((350 - 100) / 100) × 100 = %250**

---

### ROS (Return on Sales - Satış Karlılığı)

**Tanım:** Satışlardan elde edilen kar oranıdır.

**Formül:**
```
ROS = (Faaliyet Karı / Net Satışlar) × 100
```

**Örnek Hesaplama:**
- Faaliyet Karı: 800.000 TL
- Net Satışlar: 4.000.000 TL
- **ROS = (800.000 / 4.000.000) × 100 = %20**

---

## Satış Performans KPI'ları

### Sell-Through Rate (STR) / Satış Oranı

**Tanım:** Belirli bir dönemde gelen stoğun ne kadarının satıldığını gösterir.

**Formül:**
```
Sell-Through % = (Satılan Adet / (Başlangıç Stok + Alımlar)) × 100
```

veya

```
Sell-Through % = (Satılan Adet / Toplam Gelen Stok) × 100
```

**Örnek Hesaplama:**
- Başlangıç Stok: 500 adet
- Dönem Alımları: 1.500 adet
- Satılan Adet: 1.600 adet
- **Sell-Through = (1.600 / (500 + 1.500)) × 100 = %80**

**Hedef:** Sezon sonu %85+ hedeflenir.

---

### Like-for-Like (LFL) Büyüme

**Tanım:** Aynı mağazalar bazında dönemsel karşılaştırmalı büyüme oranıdır.

**Formül:**
```
LFL % = ((Bu Dönem LFL Satış - Geçen Dönem LFL Satış) / Geçen Dönem LFL Satış) × 100
```

**Örnek Hesaplama:**
- Bu Yıl (aynı mağazalar): 12.000.000 TL
- Geçen Yıl (aynı mağazalar): 10.000.000 TL
- **LFL = ((12.000.000 - 10.000.000) / 10.000.000) × 100 = %20 büyüme**

---

### Markdown % (İndirim Oranı)

**Tanım:** Orijinal fiyattan yapılan indirim yüzdesini gösterir.

**Formül:**
```
Markdown % = ((Orijinal Fiyat - İndirimli Fiyat) / Orijinal Fiyat) × 100
```

**Örnek Hesaplama:**
- Orijinal Fiyat: 500 TL
- İndirimli Fiyat: 350 TL
- **Markdown % = ((500 - 350) / 500) × 100 = %30**

---

### Average Markdown (Ortalama İndirim)

**Tanım:** Dönem boyunca uygulanan ortalama indirim oranıdır.

**Formül:**
```
Ortalama Markdown % = (Toplam İndirim Tutarı / Toplam Liste Fiyatı Satış) × 100
```

## Stok Performans KPI'ları

### GMROI (Gross Margin Return on Investment)

**Tanım:** Stok yatırımının brüt kar olarak ne kadar geri döndüğünü gösteren kritik KPI'dır.

**Formül:**
```
GMROI = (Brüt Kar / Ortalama Stok Maliyeti) × 100
```

veya

```
GMROI = ((Brüt Kar Marjı % × Stok Devir Hızı)) / 100
```

veya detaylı formül:

```
GMROI = ((Brüt Kar / Maliyet) × 52) / Stok Cover
```

**Örnek Hesaplama:**
- Brüt Kar: 600.000 TL
- Ortalama Stok Maliyeti: 200.000 TL
- **GMROI = (600.000 / 200.000) × 100 = 300%**

**Yorum:** GMROI 300% = Her 1 TL stok yatırımı için 3 TL brüt kar

**Hedef:** Minimum 200%, ideal 300%+

---

### Stock Cover (Stok Karşılama Süresi)

**Tanım:** Mevcut stokun kaç haftalık satışı karşılayacağını gösterir.

**Formül:**
```
Stock Cover = Mevcut Stok / Haftalık Ortalama Satış
```

veya adetsel:

```
Stock Cover = Mevcut Stok Adeti / 7 Gün Satış Adeti
```

**Örnek Hesaplama:**
- Mevcut Stok: 10.000 adet
- Haftalık Satış: 1.000 adet
- **Stock Cover = 10.000 / 1.000 = 10 hafta**

**Hedef:** Sezon ortası 8-12 hafta, sezon sonu 4-6 hafta

---

### Stock Turnover (Stok Devir Hızı)

**Tanım:** Stokun yıl içinde kaç kez tamamen devrettiğini gösterir.

**Formül:**
```
Stock Turnover = Satılan Malın Maliyeti (SMM) / Ortalama Stok Değeri
```

veya

```
Stock Turnover = Yıllık Satış Adeti / Ortalama Stok Adeti
```

**Örnek Hesaplama:**
- Yıllık SMM: 6.000.000 TL
- Ortalama Stok: 1.500.000 TL
- **Stock Turnover = 6.000.000 / 1.500.000 = 4 kez/yıl**

**Hedef:** Perakendede 4-6 kez/yıl

---

### Weeks of Supply (WOS)

**Tanım:** Mevcut stokun kaç hafta yeteceğini gösterir.

**Formül:**
```
WOS = Mevcut Stok / Haftalık Satış Tahmini
```

**Örnek Hesaplama:**
- Mevcut Stok: 5.000 adet
- Haftalık Tahmini Satış: 500 adet
- **WOS = 5.000 / 500 = 10 hafta**

---

### Shrinkage Rate (Fire Oranı)

**Tanım:** Kayıp, hasar, hırsızlık gibi nedenlerle oluşan stok kaybı oranıdır.

**Formül:**
```
Shrinkage % = (Kayıp Stok Değeri / Toplam Satış) × 100
```

**Örnek Hesaplama:**
- Kayıp Stok: 50.000 TL
- Toplam Satış: 5.000.000 TL
- **Shrinkage = (50.000 / 5.000.000) × 100 = %1**

**Hedef:** <%1 kabul edilebilir, <%0.5 ideal

---

## Mağaza Performans KPI'ları

### Sales per Square Meter (PSF/PSM - M² Satış)

**Tanım:** Mağaza metrekare başına satış verimliliğini ölçer.

**Formül:**
```
PSM = Toplam Satış / Mağaza Alanı (m²)
```

**Örnek Hesaplama:**
- Toplam Satış: 2.000.000 TL
- Mağaza Alanı: 150 m²
- **PSM = 2.000.000 / 150 = 13.333 TL/m²**

---

### Sales per Linear Meter (Lineer Metre Satışı)

**Tanım:** Raf/duvar uzunluğu başına satış verimliliği.

**Formül:**
```
Lineer Satış = Kategori Satışı / Kategori Lineer Metre
```

---

### Occupancy Cost Ratio (Kira/Satış Oranı)

**Tanım:** Satışların ne kadarının mağaza kirasına gittiğini gösterir.

**Formül:**
```
Occupancy Cost % = (Toplam Kira Gideri / Toplam Satış) × 100
```

**Örnek Hesaplama:**
- Yıllık Kira: 600.000 TL
- Yıllık Satış: 6.000.000 TL
- **Occupancy Cost = (600.000 / 6.000.000) × 100 = %10**

**Hedef:** <%12 ideal

---

### Labor Cost Ratio (Personel Maliyet Oranı)

**Tanım:** Satışların ne kadarının personel giderine gittiğini gösterir.

**Formül:**
```
Labor Cost % = (Toplam Personel Gideri / Toplam Satış) × 100
```

**Hedef:** <%15 ideal

---

## Müşteri KPI'ları

### UPT (Units Per Transaction - İşlem Başına Ürün)

**Tanım:** Her satış işleminde ortalama kaç ürün satıldığını gösterir.

**Formül:**
```
UPT = Toplam Satılan Adet / Toplam İşlem Sayısı
```

**Örnek Hesaplama:**
- Satılan Adet: 3.000
- İşlem Sayısı: 1.200
- **UPT = 3.000 / 1.200 = 2.5 adet/işlem**

**Hedef:** 2.0+ hedeflenir, 3.0+ mükemmel

---

### ATV (Average Transaction Value - Ortalama Sepet Tutarı)

**Tanım:** Her satış işleminin ortalama parasal değerini gösterir.

**Formül:**
```
ATV = Toplam Satış Geliri / Toplam İşlem Sayısı
```

**Örnek Hesaplama:**
- Toplam Satış: 1.500.000 TL
- İşlem Sayısı: 2.000
- **ATV = 1.500.000 / 2.000 = 750 TL**

---

### Conversion Rate (Dönüşüm Oranı)

**Tanım:** Mağazaya giren ziyaretçilerin ne kadarının satın alma yaptığını gösterir.

**Formül:**
```
Conversion % = (Satın Alan Müşteri Sayısı / Mağaza Ziyaretçi Sayısı) × 100
```

**Örnek Hesaplama:**
- Ziyaretçi: 5.000 kişi
- Satın Alan: 1.250 kişi
- **Conversion = (1.250 / 5.000) × 100 = %25**

**Hedef:** %20-30 arası normal, %30+ mükemmel

---

### ASP (Average Selling Price - Ortalama Satış Fiyatı)

**Tanım:** Satılan ürünlerin ortalama fiyatını gösterir.

**Formül:**
```
ASP = Toplam Satış Geliri / Toplam Satılan Adet
```

**Örnek Hesaplama:**
- Toplam Satış: 2.000.000 TL
- Satılan Adet: 4.000
- **ASP = 2.000.000 / 4.000 = 500 TL**

---

### Customer Retention Rate (Müşteri Tutma Oranı)

**Tanım:** Belirli dönemde tekrar alışveriş yapan müşteri oranıdır.

**Formül:**
```
Retention % = ((Dönem Sonu Müşteri - Yeni Müşteri) / Dönem Başı Müşteri) × 100
```

---

### NPS (Net Promoter Score)

**Tanım:** Müşteri sadakati ve tavsiye etme olasılığını ölçer.

**Formül:**
```
NPS = Promoter % - Detractor %
```

- **Promoter:** 9-10 puan verenler
- **Passive:** 7-8 puan verenler
- **Detractor:** 0-6 puan verenler

**Hedef:** 50+ mükemmel, 70+ dünya sınıfı

|-----|--------|-------|
