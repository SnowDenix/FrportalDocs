## Purpose

This document establishes standardized procedures for processing customer returns and exchanges across all Aydınlı Group franchise stores. Clear, consistent return policies build customer trust while protecting the business from abuse.

### Return Policy Principles

- **Customer-Centric:** Fair, transparent policies
- **Brand Protection:** Prevent return fraud and abuse
- **Consistency:** Same policy applied everywhere
- **Efficiency:** Quick processing for customer satisfaction
- **Documentation:** Full audit trail for all returns

---

## Scope

This procedure covers:
- Standard returns (refunds)
- Exchanges (same product, different size/color)
- Defective product returns
- Gift returns (no receipt)
- Online purchase returns in-store

## Process Steps

### Standard Return Policy

| Condition | Policy |
|-----------|--------|
| **Return Window** | 30 days from purchase |
| **Proof of Purchase** | Original receipt required |
| **Product Condition** | Unworn, tags attached |
| **Original Payment** | Refund to original method |
| **Without Receipt** | Exchange or store credit only |

## Standards & Rules

### Eligibility Criteria

| Criteria | Eligible | Not Eligible |
|----------|----------|--------------|
| **Timeframe** | Within 30 days | After 30 days |
| **Tags** | Attached | Removed |
| **Condition** | Unworn, clean | Worn, soiled, damaged |
| **Receipt** | Available | Missing (refund only) |
| **Category** | Regular items | Underwear, swimwear, earrings |
| **Sale Items** | Per policy | Final sale items |
| **Alterations** | None | Hemmed, tailored |

### Refund Methods

| Original Payment | Refund Method | Timing |
|------------------|---------------|--------|
| Cash | Cash | Immediate |
| Credit Card | Same card | 3-5 business days |
| Debit Card | Same card | 3-5 business days |
| Gift Card | Gift card | Immediate |
| Store Credit | Store credit | Immediate |

### Exception Handling

| Scenario | Authority | Action |
|----------|-----------|--------|
| 1-7 days past window | Supervisor | May approve |
| 8-14 days past window | Manager | May approve |
| 15+ days past window | Regional Manager | Required approval |
| No receipt, high value | Manager | Verify purchase history |
| Suspected fraud | Manager | Deny, document |

## Examples / Scenarios

### Scenario 1: Standard Return with Receipt

**Situation:** Customer returns shirt bought last week

**Process:**
```
1. Greet warmly, ask reason
2. Request receipt
3. Verify purchase date (within 30 days) ✓
4. Inspect product:
   - Tags attached ✓
   - Unworn ✓
   - No damage ✓
5. Process refund to original card
6. Provide return confirmation
7. Thank customer and invite to return
```

### Scenario 2: Return Without Receipt

**Situation:** Customer wants to return gift, no receipt

**Process:**
```
1. Greet warmly, acknowledge situation
2. Explain policy: "Without receipt, we can offer 
   exchange or store credit at current selling price"
3. If customer agrees:
   - Look up product in system
   - Issue store credit for current price
   - Process as store credit return
4. If customer disagrees:
   - Offer to check if gift-giver can provide receipt
   - Remain friendly but firm on policy
```

### Scenario 3: Past Return Window

**Situation:** Customer returns item 35 days after purchase

**Process:**
```
1. Greet warmly, check receipt
2. Note purchase date is past 30 days
3. Explain: "Our policy is 30 days, but let me see 
   what I can do"
4. Check product condition (good)
5. If within 1-7 days past:
   - Supervisor can approve
6. Offer exchange or store credit as compromise
7. Document exception if approved
```

### Scenario 4: Suspected Fraudulent Return

**Red Flags:**
- No receipt, high-value item
- Signs of wear/use
- Multiple returns from same person
- Item appears very old
- Receipt looks altered

**Process:**
```
1. Remain calm and professional
2. Conduct thorough inspection
3. If suspicious, call manager discretely
4. Manager handles conversation
5. Can deny return politely:
   "Unfortunately, this item doesn't meet our 
    return criteria as it shows signs of wear"
6. Document incident
7. Note customer details if possible
```

## Common Issues & Solutions

### Issue 1: Customer Insists on Return Outside Policy

**Solution:**
- Remain calm and empathetic
- Explain policy clearly
- Offer alternatives (exchange, store credit)
- If customer escalates, involve manager
- Manager may authorize exception if warranted
- Document any exceptions

---

### Issue 2: High Rate of "Size Doesn't Fit" Returns

**Solution:**
- Review fitting room assistance
- Train staff on size recommendations
- Ensure accurate size charts displayed
- Improve fitting room lighting
- Consider offering alterations

---

### Issue 3: Suspected Return Fraud Pattern

**Solution:**
- Track return patterns by customer
- Flag frequent returners
- Require ID for returns without receipt
- Manager review for suspicious returns
- Report patterns to Loss Prevention
