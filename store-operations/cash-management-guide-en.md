## Purpose

This document establishes comprehensive procedures for cash handling, security, and reconciliation across all Aydınlı Group franchise stores. Proper cash management protects assets, ensures accuracy, and maintains audit compliance.

### Cash Management Objectives

- **Security:** Protect cash assets from theft and loss
- **Accuracy:** Zero variance in daily reconciliation
- **Compliance:** Adhere to financial regulations
- **Accountability:** Clear audit trail for all transactions
- **Efficiency:** Streamlined cash handling processes

---

## Scope

This guide covers:
- Cash register operations
- Safe management
- Bank deposits

- Cash variance handling

## Definitions

| Term (EN) | Term (TR) | Definition |
|-----------|-----------|------------|
| Float | Açılış Nakiti | Starting cash in register at opening |

| Safe Drop | Kasaya Bırakma | Transfer of excess cash from register to safe |

| Tender | Ödeme Türü | Type of payment (cash, card, etc.) |
| Counterfeit | Sahte Para | Fake currency |
| Skim | Ara Çekim | Mid-day cash removal to safe |

### Transaction Processing

```
CASH TRANSACTION PROCEDURE

Step 1: RING TRANSACTION
├── Scan all items
├── Verify prices correct
├── Announce total to customer
└── Confirm cash payment

Step 2: RECEIVE CASH
├── Announce amount tendered
├── Check bills for authenticity
├── Keep customer's cash visible
└── Do NOT place in drawer yet

Step 3: PROCESS IN POS
├── Enter cash amount received
├── Verify change calculation
└── System calculates change

Step 4: MAKE CHANGE
├── Count change out loud
├── Count up from purchase amount
├── Hand change to customer
└── Then place received cash in drawer

Step 5: COMPLETE
├── Provide receipt
├── Thank customer
└── Close drawer immediately
```

### Safe Drop Procedure

```
When to Perform Safe Drop:
• Drawer exceeds $300 (adjustable threshold)
• Before lunch break
• During slow periods
• Any time drawer feels "heavy"

Safe Drop Steps:
1. Alert supervisor/manager
2. Print X-Report (optional)
3. Count excess cash
4. Complete safe drop slip
5. Place cash in deposit envelope
6. Have witness verify amount
7. Drop in safe
8. Retain copy of slip
9. Record in log book
```

## Standards & Rules

### Cash Handling Standards

| Standard | Requirement | Consequence |
|----------|-------------|-------------|
| Drawer access | Assigned user only | Disciplinary action |
| Cash visibility | Always count in view | Retraining |
| Drawer status | Closed when not in use | Verbal warning |
| Large bills | Verify with detector pen | Counterfeit acceptance |
| Variance tolerance | ±$5 per day | Investigation required |

### Float Standards by Store

| Store Type | Opening Float | Safe Drop Threshold |
|------------|---------------|---------------------|
| Small | $200 | $300 |
| Medium | $300 | $500 |
| Large | $500 | $750 |
| Flagship | $750 | $1,000 |

### Denomination Requirements

| Denomination | Opening Quantity | Notes |
|--------------|------------------|-------|
| $100 | 0 | Accept but don't hold |
| $50 | 0 | Accept but don't hold |
| $20 | 5 ($100) | Primary bill |
| $10 | 5 ($50) | Change making |
| $5 | 10 ($50) | Change making |
| $1 | 25 ($25) | Change making |
| Quarters | $10 roll | Coins |
| Dimes | $5 roll | Coins |
| Nickels | $2 roll | Coins |
| Pennies | $0.50 roll | Coins |

### Prohibited Practices

- ❌ Leaving drawer open unattended
- ❌ Sharing drawer access codes
- ❌ Personal transactions on duty
- ❌ IOU or personal loans from drawer
- ❌ Cash advances
- ❌ Making change without transaction
- ❌ Counting cash where customers can see amounts
- ❌ Discussing cash amounts publicly

## Examples / Scenarios

### Scenario 1: Cash Variance Investigation

**Situation:** $20 short at closing

**Investigation Steps:**
```
1. RECOUNT
   ├── Remove all cash from drawer
   ├── Count by denomination
   ├── Have witness verify
   └── If still short, proceed

2. REVIEW TRANSACTIONS
   ├── Print detailed Z-report
   ├── Review all voids
   ├── Review all returns
   ├── Check for missing transactions
   └── Look for common errors

3. PHYSICAL CHECK
   ├── Check under drawer
   ├── Check around register
   ├── Check in trash carefully
   └── Check pockets (staff volunteer)

4. DOCUMENT
   ├── Complete variance report
   ├── Manager review and sign
   ├── Forward to Regional Manager
   └── File for records

5. FOLLOW-UP
   ├── Review CCTV if available
   ├── Interview staff if needed
   └── Implement preventive measures
```

### Scenario 2: Suspected Counterfeit Bill

**Situation:** Customer pays with suspicious $100 bill

**Response Protocol:**
1. Discreetly test with detector pen
2. Check security features
3. If suspicious:
   - Do NOT return bill
   - Politely ask for alternative payment
   - "I'm sorry, our system won't accept this bill"
   - Contact manager
   - Complete counterfeit report
   - Contact authorities if required

### Scenario 3: Customer Disputes Change

**Situation:** Customer claims received wrong change

**Response Protocol:**
1. Remain calm and professional
2. Apologize for any confusion
3. Ask manager to witness
4. Offer to count drawer (requires closing)
5. If error found, correct immediately
6. If no error, explain politely
7. Document incident

## Common Issues & Solutions

### Issue 1: Frequent Small Variances

**Problem:** $1-5 variances occurring regularly

**Solutions:**
- Review transaction handling
- Retrain on counting change
- Check for calculation errors
- Verify POS is calculating correctly
- Implement counting out loud

---

### Issue 2: Register Balances Taking Too Long

**Problem:** Closing takes over 30 minutes

**Solutions:**
- Organize denominations throughout day
- Perform mid-day X-report checks
- Use counting trays/mats
- Better safe drop discipline
- Streamline paperwork

---

### Issue 3: Counterfeit Bills Accepted

**Problem:** Fake currency found during deposit

**Solutions:**
- Train all staff on detection
- Provide detection pens/devices
- Verify all bills $20 and above
- Post security features reference
- Immediate reporting protocol
