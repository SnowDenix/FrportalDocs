## Purpose

This document establishes comprehensive procedures for inventory management, stock control, and cycle counting across all Aydınlı Group franchise stores. Accurate inventory ensures product availability, minimizes shrinkage, and optimizes working capital.

### Inventory Management Objectives

- **Accuracy:** 99%+ inventory accuracy
- **Availability:** Right product, right place, right time
- **Efficiency:** Optimized stock levels
- **Visibility:** Real-time inventory knowledge
- **Protection:** Minimize shrinkage and loss

---

## Scope

This guide covers:
- Receiving and processing deliveries
- Stock organization and replenishment
- Cycle counting procedures
- Annual inventory counts
- Inventory adjustment processes
- Inter-store transfers

### Cycle Count Process

```
CYCLE COUNT SCHEDULE

| Day | Category | Example |
|-----|----------|---------|
| Monday | Tops | Polos, shirts, t-shirts |
| Tuesday | Bottoms | Pants, shorts, jeans |
| Wednesday | Outerwear | Jackets, coats |
| Thursday | Accessories | Belts, ties, bags |
| Friday | Catch-up | Any incomplete areas |

CYCLE COUNT STEPS:
1. Print count sheet for assigned area
2. Clear area of customers if possible
3. Count actual items on floor and back
4. Record count on sheet
5. Compare to system quantity
6. Investigate any variances
7. Submit count results
8. Adjust inventory if verified
```

## Examples / Scenarios

### Scenario 1: Delivery with Shortage

**Situation:** Delivery of 100 units, packing slip shows 120

**Process:**
```
1. Document shortage immediately
2. Note on delivery receipt: "Received 100, expected 120"
3. Have driver sign amended receipt
4. Complete discrepancy report
5. Receive only 100 in system
6. Contact HQ with details
7. File all paperwork
8. Follow up on resolution
```

### Scenario 2: Cycle Count Variance Found

**Situation:** System shows 15, actual count is 12

**Process:**
```
1. Recount to verify
2. Check fitting rooms
3. Check under fixtures
4. Check back stock
5. Review recent sales
6. Check for mis-scans
7. If variance confirmed:
   - Document findings
   - Manager review
   - Adjust inventory if approved
   - Note reason code
8. If pattern found, investigate further
```

### Scenario 3: Damaged Goods Received

**Situation:** 5 items in delivery have defects

**Process:**
```
1. Separate damaged items
2. Document with photos
3. Note on receiving paperwork
4. Do NOT put in inventory
5. Complete damaged goods form
6. Submit to HQ
7. Hold items per policy:
   - Return to vendor, OR
   - Mark down and sell, OR
   - Destroy with documentation
8. Update inventory accordingly
```

## Common Issues & Solutions

### Issue 1: Chronic Stock-outs

**Problem:** Same items repeatedly out of stock

**Solutions:**
- Review reorder points
- Analyze sales velocity
- Increase order quantities
- Request priority allocation
- Consider store transfer

---

### Issue 2: High Shrinkage Rate

**Problem:** Inventory loss exceeding targets

**Solutions:**
- Increase security measures
- Review receiving procedures
- Conduct more cycle counts
- Train staff on loss prevention
- Analyze shrinkage patterns

---

### Issue 3: Disorganized Back Stock

**Problem:** Can't find items when needed

**Solutions:**
- Implement zone system
- Daily organization time
- Clear labeling
- Regular purging of old items
- Staff accountability
