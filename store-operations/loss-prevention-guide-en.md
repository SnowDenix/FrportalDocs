## Purpose

This document establishes comprehensive procedures for preventing, detecting, and responding to theft and inventory shrinkage across all Aydınlı Group franchise stores. Effective loss prevention protects company assets while maintaining a positive shopping environment.

### Loss Prevention Objectives

- **Prevention:** Deter theft through visible measures
- **Detection:** Identify theft when it occurs
- **Response:** Handle incidents professionally
- **Recovery:** Maximize asset recovery
- **Culture:** Create shrinkage-aware environment

---

### Shoplifter Profile Awareness

```
WARNING SIGNS (Not all indicate theft)

BEHAVIORAL SIGNALS:
• Avoiding eye contact
• Nervous behavior
• Excessive browsing without buying
• Watching staff more than merchandise
• Wearing loose/oversized clothing
• Carrying large bags
• Entering fitting room multiple times
• Staying close to exits
• Shopping in groups that split up

REMEMBER:
- These are indicators only, NOT proof
- Never accuse without evidence
- Focus on customer service approach
- Many innocent customers show these behaviors
```

### EAS Gate Alarm Response

```
WHEN EAS ALARM SOUNDS:

1. ACKNOWLEDGE
   └── Make eye contact with customer
   └── Smile and approach professionally

2. APPROACH
   └── "Excuse me, it seems the alarm went off.
        This sometimes happens. May I check your bag?"

3. CHECK
   └── Visually inspect bags (customer holds open)
   └── Check receipt against items
   └── Look for missed security tags
   └── Check under shopping bag

4. IF TAG FOUND
   └── "I apologize - we missed removing this tag.
        Let me take care of that for you."
   └── Remove tag
   └── Thank customer

5. IF NO TAG FOUND
   └── "I apologize for the inconvenience.
        Have a great day!"
   └── Document false alarm

6. IF THEFT SUSPECTED
   └── Do NOT accuse
   └── Call manager discretely
   └── Let customer leave if manager unavailable
   └── Document and review CCTV
```

## Examples / Scenarios

### Scenario 1: Observed Shoplifting

**Situation:** Staff sees customer conceal item in bag

**Response:**
```
DO:
✓ Stay calm
✓ Maintain visual contact
✓ Signal to colleague/manager
✓ Continue customer service approach
✓ Note description details
✓ Let manager handle

DON'T:
✗ Accuse the customer
✗ Physically confront
✗ Block exit
✗ Chase outside store
✗ Put yourself at risk

IF CUSTOMER EXITS:
1. Do NOT follow outside
2. Note direction of travel
3. Document immediately
4. Save CCTV footage
5. Call police if high value
```

### Scenario 2: Fitting Room Discrepancy

**Situation:** Customer entered with 4 items, exits with 2

**Response:**
```
1. "Excuse me, I believe you may have left some items
    in the fitting room. Let me check for you."

2. Check fitting room together

3. If items found:
   - Return items to customer
   - "Here are the other items you were trying."

4. If items NOT found:
   - "I'm sorry, I must have miscounted."
   - Do NOT accuse
   - Let customer leave
   - Review CCTV immediately
   - Document incident
```

### Scenario 3: Suspected Employee Theft

**Situation:** Cash shortages or inventory variances

**Response:**
```
IMPORTANT: Handle confidentially

1. Document all facts
2. Review CCTV footage
3. Analyze patterns (dates, shifts)
4. Contact Regional Manager
5. Do NOT confront employee
6. Let HR/LP team investigate
7. Maintain normal operations
8. Preserve evidence
```

## Common Issues & Solutions

### Issue 1: High Fitting Room Theft

**Problem:** Items going missing from fitting rooms

**Solutions:**
- Stricter item counting
- Lower item limits
- More frequent room checks
- Remove empty hangers
- Consider attendant position

---

### Issue 2: EAS Tags Being Defeated

**Problem:** Tags removed or bypassed

**Solutions:**
- Use harder tags on high-risk items
- Multiple tags per item
- Vary tag placement
- Regular staff training

---

### Issue 3: High Internal Theft

**Problem:** Employee theft suspected

**Solutions:**
- Strengthen cash handling procedures
- Implement bag checks
- Review CCTV regularly
- Background checks on new hires
- Anonymous reporting line
