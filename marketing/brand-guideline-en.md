## Purpose

This document establishes the official brand guidelines for U.S. Polo Assn., Pierre Cardin, and Cacharel brands within the Aydınlı Group franchise network. It ensures consistent brand representation across all touchpoints, protecting brand equity and delivering a unified customer experience worldwide.

### Brand Management Objectives

| Objective | Description |
|-----------|-------------|
| **Consistency** | Ensure uniform brand presentation across all markets |
| **Protection** | Safeguard brand integrity and equity |
| **Recognition** | Build strong brand awareness through consistent visuals |
| **Differentiation** | Maintain distinct brand identities within the portfolio |
| **Compliance** | Ensure adherence to licensor requirements |

## Definitions

| Term | Definition |
|------|------------|
| **Brand Identity** | The collection of visual and verbal elements that represent a brand |
| **Logo** | The primary graphic symbol representing the brand |
| **Logotype** | The brand name rendered in a specific typeface |
| **Brand Mark** | A symbol or icon used alongside or instead of the logotype |
| **Clear Space** | Minimum required empty space around the logo |
| **Primary Colors** | The main colors in a brand's color palette |
| **Secondary Colors** | Supporting colors used alongside primary colors |
| **Typography** | The fonts and typefaces approved for brand use |
| **Tone of Voice** | The personality and style of brand communications |
| **Brand Architecture** | The organization and hierarchy of brands within a portfolio |
| **Co-branding** | Combining two or more brands in a single communication |
| **Licensor** | The brand owner granting usage rights |

## Logo Usage Rules

### Universal Rules (All Brands)

#### Permitted Logo Usage

✓ Use only official logo files from Brand Portal
✓ Maintain required clear space
✓ Use approved color variations only
✓ Respect minimum size requirements
✓ Position on contrasting backgrounds

#### Prohibited Logo Usage

✗ Do not stretch or distort the logo
✗ Do not rotate the logo
✗ Do not add effects (shadows, gradients, outlines)
✗ Do not change logo colors
✗ Do not place on busy backgrounds
✗ Do not rearrange logo elements
✗ Do not recreate or redraw the logo

### Logo Background Guidelines

| Background Type | Recommendation |
|-----------------|----------------|
| Light solid colors | Use primary color logo |
| Dark solid colors | Use white/reversed logo |
| Photographs | Use white logo with shadow or box |
| Patterns | Avoid or use solid color box |

### Co-branding Rules

1. Never place competing brand logos together
2. Aydınlı corporate logo separate from consumer brands
3. Mall/location logos in designated areas only
4. Event/sponsor logos require HQ approval

## Common Issues & Solutions

### Issue 1: Logo Appearing Distorted

**Symptom:** Logo looks stretched, squashed, or pixelated.

**Causes:**
- Wrong file format used
- Manual resizing without proportions
- Low-resolution file for print

**Solution:**
1. Always use vector files (AI, EPS, SVG) for print
2. Use PNG or SVG for digital
3. Lock aspect ratio when resizing
4. Download correct file from Brand Portal

---

### Issue 2: Color Mismatch in Print

**Symptom:** Printed colors don't match brand standards.

**Causes:**
- Incorrect color profile
- Monitor not calibrated
- Printer using wrong ink settings

**Solution:**
1. Provide Pantone reference to printer
2. Request color proof before full production
3. Use approved print vendors when possible
4. Keep physical color swatch for reference

---

### Issue 3: Unauthorized Brand Usage

**Symptom:** Non-compliant materials appearing in stores.

**Causes:**
- Local vendors creating without guidelines
- Staff using personal designs
- Outdated materials still in use

**Solution:**
1. Provide Brand Guidelines to all vendors
2. Conduct regular compliance audits
3. Remove unauthorized materials immediately
4. Re-train staff on brand standards

---

### Issue 4: Missing Brand Assets

**Symptom:** Cannot find required logo or asset files.

**Causes:**
- Brand Portal access issues
- Files not yet uploaded
- Regional restrictions

**Solution:**
1. Check Brand Portal first
2. Contact Regional Marketing Manager
3. Request specific assets via support ticket
4. Document asset needs for future planning
