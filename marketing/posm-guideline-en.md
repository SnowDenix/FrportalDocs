## Purpose

This document provides comprehensive guidelines for Point of Sale Materials (POSM) management across all Aydınlı Group franchise locations. It ensures consistent, high-quality visual communication at the point of purchase, supporting brand image and driving sales.

### POSM Objectives

| Objective | Description |
|-----------|-------------|
| **Visibility** | Ensure maximum campaign and product visibility |
| **Consistency** | Uniform presentation across all stores |
| **Impact** | Create compelling visual displays that drive sales |
| **Compliance** | Meet brand standards and regulatory requirements |
| **Efficiency** | Optimize production, distribution, and installation |

## Definitions

| Term | Definition |

## Installation Guidelines

### Pre-Installation Preparation

**Materials Needed:**
- POSM kit components
- Cleaning supplies
- Installation tools (tape, hooks, stands)
- Measuring tape
- Level (for alignment)
- Ladder (for high placements)

**Preparation Steps:**
1. Review POSM kit contents against packing list
2. Inspect all materials for damage
3. Clean installation surfaces
4. Remove previous campaign materials
5. Plan placement according to guidelines

### Installation Process by POSM Type

**Window Posters:**
```
1. Clean window interior surface
2. Measure and mark placement points
3. Apply poster (tape or cling method)
4. Ensure no bubbles or wrinkles
5. Check alignment from outside
6. Verify visibility from 5+ meters away
```

**Ceiling Danglers:**
```
1. Identify suspension points (existing hooks/rails)
2. Attach fishing line or provided hanging mechanism
3. Ensure dangler hangs level
4. Position above product display area
5. Check for adequate clearance (min 2.2m from floor)
6. Secure any moving parts
```

**Floor Stickers:**
```
1. Clean floor surface thoroughly
2. Allow floor to dry completely
3. Position sticker in exact location
4. Apply from one edge, smoothing outward
5. Eliminate all air bubbles
6. Apply edge sealant if provided
7. Allow 1 hour before foot traffic
```

**Counter Displays:**
```
1. Clear counter space
2. Assemble display per instructions
3. Position at optimal customer angle
4. Ensure stability (won't tip)
5. Add any product samples if included
6. Check visibility from customer approach angle
```

### Installation Quality Checklist

- [ ] All POSM components accounted for
- [ ] Surfaces cleaned before installation
- [ ] Placement matches guideline specifications
- [ ] Materials are level and aligned
- [ ] No bubbles, wrinkles, or damage visible
- [ ] Messaging is correct and legible
- [ ] Brand colors and logos appear accurate
- [ ] Safety requirements met (no trip hazards)
- [ ] Before/after photos taken

|-----------|--------|----------|

## Standards & Rules

### Mandatory POSM Rules

1. **Installation:** All campaign POSM must be installed by launch date
2. **Completeness:** All kit items must be displayed (no partial execution)
3. **Placement:** Follow exact placement guidelines provided
4. **Quality:** No damaged or defaced POSM to be displayed
5. **Currency:** Only current campaign POSM to be shown
6. **Removal:** Previous campaign POSM removed before new installation
7. **Safety:** All safety standards must be met

### Prohibited Actions

- Modifying POSM design or messaging
- Installing POSM in unauthorized locations
- Using POSM from other brands or campaigns
- Keeping expired POSM on display
- Damaging or defacing POSM
- Using personal or handwritten signage
- Blocking safety features with POSM

### POSM Disposal Guidelines

| Material Type | Disposal Method |
|---------------|-----------------|
| Paper/cardboard | Recycle |
| Vinyl/plastic | Dispose per local regulations |
| Foamboard | Dispose per local regulations |
| Reusable frames | Return to warehouse |
| Hardware (hooks, stands) | Keep for reuse |

## Common Issues & Solutions

### Issue 1: POSM Arrived Damaged

**Symptom:** Materials received with visible damage.

**Actions:**
1. Document damage with photos
2. Note on delivery receipt
3. Report to Regional Manager within 24 hours
4. Request replacement immediately
5. Use digital alternatives temporarily if available

---

### Issue 2: POSM Doesn't Fit Store Layout

**Symptom:** Standard POSM size doesn't work in store.

**Solutions:**
1. Review placement guidelines for alternatives
2. Request custom-sized POSM (4+ weeks lead time)
3. Use smaller alternative from kit
4. Consult Regional Manager for approved adjustments

---

### Issue 3: Floor Stickers Peeling

**Symptom:** Edges of floor stickers lifting or peeling.

**Prevention:**
- Clean and dry floor before application
- Apply edge sealant
- Avoid high-traffic corners if possible

**Solutions:**
1. Re-seal edges with approved adhesive
2. Replace if significantly damaged
3. Remove if safety hazard

---

### Issue 4: Inconsistent Installation Across Stores

**Symptom:** Regional audit shows varying POSM quality.

**Solutions:**
1. Conduct retraining session
2. Provide photographic examples
3. Implement installation checklist
4. Schedule follow-up compliance check
