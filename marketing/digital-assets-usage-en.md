## Purpose

This document provides comprehensive guidelines for accessing, using, and managing digital marketing assets for U.S. Polo Assn., Pierre Cardin, and Cacharel brands within the Aydınlı Grup franchise network. Proper usage of digital assets ensures brand consistency across all digital touchpoints.

### Digital Asset Management Objectives

| Objective | Description |
|-----------|-------------|
| **Consistency** | Ensure uniform brand representation across digital channels |
| **Accessibility** | Easy access to approved assets for all authorized users |
| **Quality** | Maintain high-quality standards for all digital materials |
| **Compliance** | Ensure adherence to brand guidelines and licensing terms |
| **Efficiency** | Reduce time spent searching for and creating assets |

## Definitions

| Term | Definition |
|------|------------|
| **DAM (Digital Asset Management)** | System for storing, organizing, and distributing digital assets |
| **Brand Portal** | Central repository for all brand-approved digital assets |
| **Master File** | Original high-resolution file from which derivatives are created |
| **Derivative** | Adapted version of master file for specific use (e.g., web, print) |
| **Aspect Ratio** | Proportional relationship between width and height of an image |
| **Resolution** | Image quality measured in pixels (digital) or DPI (print) |
| **Color Profile** | Color space specification (RGB for digital, CMYK for print) |
| **File Format** | Digital file type (JPG, PNG, SVG, MP4, etc.) |
| **Metadata** | Embedded information about the file (title, keywords, usage rights) |
| **Watermark** | Visual mark indicating preview status or ownership |
| **Rights Managed** | Assets with specific licensing terms and usage restrictions |
| **Royalty Free** | Assets licensed for unlimited use within terms |

## Usage Guidelines

### General Usage Rules

1. **Always use latest version** - Check for asset updates before use
2. **Maintain aspect ratio** - Never stretch or distort images
3. **Respect safe zones** - Keep clear space around logos
4. **Use appropriate resolution** - Match resolution to output
5. **Follow color profiles** - RGB for digital, CMYK for print

### Permitted Usage

✓ Official franchise marketing materials
✓ Approved social media channels
✓ Store-level promotional materials (with approval)
✓ Internal presentations
✓ Approved website and digital platforms
✓ Email marketing through approved systems

### Prohibited Usage

✗ Personal use
✗ Unauthorized third-party platforms
✗ Modification without approval
✗ Resale or redistribution
✗ Use in non-brand contexts
✗ Combining with competitor brands
✗ Use after license termination

### Image Usage Restrictions

| Asset Type | Restrictions |
|------------|-------------|
| **Model Photography** | Limited to franchise marketing only; no modification of model appearance |
| **Licensed Imagery** | Check individual licensing terms; some images time-limited |
| **Campaign-Specific** | Use only during active campaign period |
| **Seasonal** | Replace with current season assets when available |

## Standards & Rules

### Quality Standards

| Aspect | Standard |
|--------|----------|
| Resolution | Minimum 72 DPI for digital, 300 DPI for print |
| File Integrity | No artifacts, corruption, or visible compression |
| Color Accuracy | Must match brand color specifications |
| Composition | Follow approved cropping and safe zones |
| Naming | Follow naming conventions strictly |

### Compliance Rules

1. **Version Control:** Always use the most current asset version
2. **Rights Verification:** Check licensing before use
3. **Approval Trail:** Maintain records of approvals for custom work
4. **Expiration Dates:** Remove assets when campaigns end
5. **Attribution:** Credit photographers/creators when required

### Storage and Organization

| Requirement | Standard |
|-------------|----------|
| Local Storage | Organize by brand → campaign → asset type |
| Backup | Do not store as sole copy; use Brand Portal |
| Deletion | Remove campaign assets within 30 days of end |
| Sharing | Use Brand Portal sharing, not email attachments |

## Common Issues & Solutions

### Issue 1: Cannot Find Required Asset

**Symptom:** Asset not appearing in search results.

**Solutions:**
1. Check spelling and try alternative keywords
2. Verify you have appropriate access level
3. Check if asset exists in Archive folder
4. Contact Regional Marketing for assistance
5. Submit custom asset request if truly unavailable

---

### Issue 2: Downloaded Image Quality Poor

**Symptom:** Image appears pixelated or blurry.

**Causes:**
- Downloaded wrong size/resolution
- Enlarging beyond native resolution
- Incorrect format for use case

**Solutions:**
1. Download higher resolution version
2. Use appropriate size for intended output
3. For print, ensure 300 DPI minimum
4. Request higher quality source if unavailable

---

### Issue 3: Asset Appears Different When Used

**Symptom:** Colors or appearance don't match preview.

**Causes:**
- Color profile mismatch
- Monitor not calibrated
- Incorrect file format

**Solutions:**
1. Verify RGB for digital, CMYK for print
2. Calibrate monitor for accurate preview
3. Download correct format for use case
4. Contact HQ if asset file is incorrect

---

### Issue 4: Cannot Access Brand Portal

**Symptom:** Login fails or access denied.

**Solutions:**
1. Verify username and password
2. Check with Regional Manager for access status
3. Clear browser cache and cookies
4. Try different browser
5. Contact IT support if issues persist
