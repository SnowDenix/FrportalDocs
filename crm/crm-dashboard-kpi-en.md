## CRM KPIs

### 1. Customer Matching KPIs

#### Loyalty Penetration Rate

**Definition:** Percentage of total sales transactions matched with loyalty members

**Formula:**
```
Loyalty Penetration % = (Member Sales Count / Total Sales Count) Ã— 100
```

**Example Calculation:**
- Total sales count: 1,000
- Member sales count: 720
- Loyalty Penetration = (720 / 1,000) Ã— 100 = **72%**

**Target:** 70% and above

**Franchise Impact:** High penetration = more customer data = better targeting

---

#### Customer Match Rate

**Definition:** Sales matched to registered CRM customers

**Formula:**
```
Customer Match Rate % = (Matched Sales / Total Sales) Ã— 100
```

**Target:** 75% and above

---

### 2. Customer Base KPIs

#### Active Customer Rate

**Definition:** Ratio of customers who purchased in last 12 months to total member base

**Formula:**
```
Active Customer Rate % = (Last 12 Month Active Customers / Total Members) Ã— 100
```

**Example Calculation:**
- Total members: 50,000
- Purchased in last 12 months: 20,000
- Active Customer Rate = (20,000 / 50,000) Ã— 100 = **40%**

**Target:** 35% and above

**Franchise Impact:** Active customer rate shows loyalty program effectiveness

---

#### New Customer Acquisition

**Definition:** Number of new customers acquired during the period

**Formula:**
```
New Customer Rate % = (New Customers / Total Customers with Sales) Ã— 100
```

**Target:** 20-30% range (varies by sector and location)

---

#### Customer Retention Rate

**Definition:** Rate of customers who purchased in previous period purchasing again in current period

**Formula:**
```
Retention Rate % = ((Period End Active - New Acquired) / Period Start Active) Ã— 100
```

**Example Calculation:**
- Period start active: 10,000
- Period end active: 11,000
- New acquired: 3,000
- Retention = ((11,000 - 3,000) / 10,000) Ã— 100 = **80%**

**Target:** 70% and above

---

### 3. Customer Value KPIs

#### Average Transaction Value (ATV)

**Definition:** Average basket value

**Formula:**
```
ATV = Total Sales Amount / Total Transaction Count
```

**Example Calculation:**
- Total sales: 500,000
- Total transactions: 1,000
- ATV = 500,000 / 1,000 = **500**

---

#### Units Per Transaction (UPT)

**Definition:** Number of items sold per transaction

**Formula:**
```
UPT = Total Units Sold / Total Transaction Count
```

**Example Calculation:**
- Units sold: 2,500
- Transactions: 1,000
- UPT = 2,500 / 1,000 = **2.5 units**

**Target:** 2.0 and above

---

#### Customer Lifetime Value (CLV)

**Definition:** Estimated lifetime revenue from a customer

**Formula:**
```
CLV = Average Order Value Ã— Annual Purchase Frequency Ã— Customer Lifespan (years)
```

**Example Calculation:**
- Average order: 500
- Annual frequency: 4 times
- Customer lifespan: 5 years
- CLV = 500 Ã— 4 Ã— 5 = **10,000**

---

### 4. Loyalty Program KPIs

#### Points Redemption Rate

**Definition:** Rate of earned points being used

**Formula:**
```
Redemption Rate % = (Points Used / Points Earned) Ã— 100
```

**Example Calculation:**
- Points earned: 1,000,000
- Points used: 350,000
- Redemption Rate = (350,000 / 1,000,000) Ã— 100 = **35%**

**Target:** 25-40% range (too low = program unknown, too high = cost)

---

#### Coupon Redemption Rate

**Definition:** Rate of distributed coupons being used

**Formula:**
```
Coupon Redemption % = (Coupons Used / Coupons Distributed) Ã— 100
```

**Target:** 15-25% range

---

#### Enrollment Rate

**Definition:** Rate of new customers registering to loyalty program

**Formula:**
```
Enrollment Rate % = (New Members / New Customers) Ã— 100
```

**Target:** 50% and above

---

### 5. Campaign KPIs

#### Campaign Response Rate

**Definition:** Campaign target audience participation rate

**Formula:**
```
Response Rate % = (Campaign Participants / Target Audience) Ã— 100
```

**Target:** 10-20% range

---

#### Campaign ROI

**Definition:** Campaign return on investment

**Formula:**
```
ROI = ((Campaign Revenue - Campaign Cost) / Campaign Cost) Ã— 100
```

**Example Calculation:**
- Campaign revenue: 100,000
- Campaign cost: 20,000
- ROI = ((100,000 - 20,000) / 20,000) Ã— 100 = **400%**

**Target:** 200% and above

---

### 6. Customer Acquisition KPIs

#### Customer Acquisition Cost (CAC)

**Definition:** Cost to acquire a new customer

**Formula:**
```
CAC = Total Marketing Expense / Number of New Customers Acquired
```

**Example Calculation:**
- Marketing expense: 50,000
- New customers: 500
- CAC = 50,000 / 500 = **100/customer**

**Target:** Around 10-20% of CLV

---

### 7. Ticketing KPIs

#### SLA Compliance Rate

**Definition:** Rate of tickets closed within SLA

**Formula:**
```
SLA Compliance % = (Closed Within SLA / Total Closed) Ã— 100
```

**Target:** 95% and above

---

#### First Contact Resolution (FCR)

**Definition:** Rate of tickets resolved on first contact

**Formula:**
```
FCR % = (Resolved on First Contact / Total Tickets) Ã— 100
```

**Target:** 60% and above

## Common Issues & Solutions

### Issue 1: KPI Data Not Current

**Situation:** Dashboard data appears not updated.

**Solution:**
1. Refresh the page
2. Clear cache
3. Wait 15-30 minutes (data synchronization)
4. If problem persists, report to IT

---

### Issue 2: Low Loyalty Penetration

**Situation:** Penetration rate below target.

**Possible Causes:**
- Staff not matching customers
- Insufficient new registration incentive
- System usage difficulty

**Solution:**
1. Set daily penetration target
2. Explain importance of matching to staff
3. Ensure customer query at every sale

---

### Issue 3: Very Low Redemption Rate

**Situation:** Customers not using their points.

**Possible Causes:**
- Customer unaware of points balance
- Difficult usage conditions
- Staff not reminding

**Solution:**
1. Remind points balance at every sale
2. Encourage points usage
3. Send points reminder via SMS
