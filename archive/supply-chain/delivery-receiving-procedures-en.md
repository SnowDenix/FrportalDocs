## Purpose

This document establishes standardized procedures for receiving deliveries at franchise stores. Proper receiving ensures inventory accuracy, minimizes losses, and maintains product quality from distribution center to sales floor.

### Key Objectives

| Objective | Impact |
|-----------|--------|
| **Accuracy** | 100% inventory accuracy |
| **Efficiency** | Minimize receiving time |
| **Quality** | Protect product condition |
| **Documentation** | Complete record keeping |
| **Loss Prevention** | Identify discrepancies immediately |

## Process Steps

### Standard Receiving Process

```
┌────────────────────────────────────────────────────────────────┐
│                  DELIVERY RECEIVING PROCESS                    │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   PHASE 1: PRE-DELIVERY PREPARATION                            │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ • Review ASN / expected delivery notice                 │  │
│   │ • Clear receiving area                                  │  │
│   │ • Prepare equipment (scanner, cart, tools)              │  │
│   │ • Print expected items list                             │  │
│   │ • Assign receiving staff                                │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   PHASE 2: DELIVERY ARRIVAL                                    │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ • Verify carrier/vehicle identification                 │  │
│   │ • Check seal integrity (if sealed)                      │  │
│   │ • Document seal number                                  │  │
│   │ • Inspect external condition of packages                │  │
│   │ • Count total number of packages/boxes                  │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   PHASE 3: PACKAGE VERIFICATION                                │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ • Compare box count to delivery documents               │  │
│   │ • Note any visible external damage                      │  │
│   │ • Photograph damaged packages before opening            │  │
│   │ • Sign delivery receipt with notes                      │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   PHASE 4: DETAILED RECEIVING                                  │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ • Open each package systematically                      │  │
│   │ • Scan/count all items                                  │  │
│   │ • Verify against packing list                           │  │
│   │ • Inspect item condition                                │  │
│   │ • Separate any damaged items                            │  │
│   │ • Document discrepancies immediately                    │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   PHASE 5: DOCUMENTATION & SYSTEM UPDATE                       │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ • Complete receiving form                               │  │
│   │ • File discrepancy report (if any)                      │  │
│   │ • Update inventory system                               │  │
│   │ • Process items to stock                                │  │
│   │ • Archive all documentation                             │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Discrepancy Handling Process

```
DISCREPANCY IDENTIFIED

├── SHORTAGE (Missing Items)
│   ├── Document on receiving form
│   ├── Note on BOL before signing
│   ├── Do NOT adjust inventory for missing items
│   ├── Complete shortage claim form
│   ├── Submit to HQ within 24 hours
│   └── Await credit/replacement

├── OVERAGE (Extra Items)
│   ├── Document on receiving form
│   ├── Contact HQ for instructions
│   ├── Do NOT add to inventory until approved
│   ├── Secure items separately
│   └── Follow HQ direction

├── DAMAGE (Damaged Items)
│   ├── Photograph all damage
│   ├── Document on receiving form
│   ├── Separate from good inventory
│   ├── Complete damage claim form
│   ├── Hold for inspection if requested
│   └── Submit claim within 24 hours

└── WRONG ITEMS (Incorrect Products)
    ├── Document on receiving form
    ├── Do NOT add to inventory
    ├── Contact HQ immediately
    ├── Hold items for return
    └── Follow return instructions
```

## Examples / Scenarios

### Scenario 1: Normal Delivery

**Situation:** Regular weekly replenishment arrives

**Process:**
```
1. Verify carrier and seal number
2. Count boxes (12 expected, 12 received)
3. Sign BOL
4. Open and scan all items
5. All items match packing list
6. Update inventory system
7. Process items to sales floor
8. File documentation
```

**Result:** Clean receiving, no discrepancies

---

### Scenario 2: Delivery with Shortage

**Situation:** 5 items missing from shipment

**Process:**
```
1. Verify carrier and complete box count
2. Open and scan all items
3. Discover shortage: 95 received vs 100 expected
4. Check all boxes for missing items
5. Document shortage on BOL before signing
6. Complete receiving for items present
7. Complete discrepancy report
8. Photograph documentation
9. Submit to HQ within 24 hours
10. Do NOT adjust inventory for missing items
11. Await HQ resolution
```

**Result:** Shortage documented, claim filed

---

### Scenario 3: Delivery with External Damage

**Situation:** Box arrives visibly damaged

**Process:**
```
1. Photograph damaged box before signing
2. Note "RECEIVED DAMAGED" on BOL
3. Have carrier sign acknowledgment
4. Open damaged box carefully
5. Inspect all items inside
6. Document any product damage
7. Separate damaged items
8. Complete damage claim form
9. Submit claim with photos
10. Hold damaged items for inspection
```

**Result:** Damage documented, carrier notified

---

### Scenario 4: Delivery with Wrong Items

**Situation:** Received items not on order

**Process:**
```
1. Verify items against packing list
2. Confirm items do not belong to store
3. Do NOT scan into inventory
4. Contact HQ immediately
5. Document wrong items received
6. Secure items separately
7. Await return instructions
8. Arrange pickup/return
```

**Result:** Wrong items returned to DC

## Common Issues & Solutions

### Issue 1: Delivery During Peak Hours

**Problem:** Delivery arrives during busy customer time

**Solutions:**
- Schedule delivery windows in advance
- Communicate preferred times to carrier
- Have dedicated receiving staff
- Use back room for quick holding
- Process after peak hours if needed

---

### Issue 2: Scanner Malfunction

**Problem:** Cannot scan items during receiving

**Solutions:**
- Have backup scanner available
- Manual count as alternative
- Document manually and update later
- Contact IT support
- Never delay receiving completion

---

### Issue 3: Incomplete Documentation

**Problem:** Packing list missing or unclear

**Solutions:**
- Request documentation from carrier
- Contact HQ for packing list
- Perform blind count
- Document all items received
- Note missing documentation on BOL

---

### Issue 4: Storage Constraints

**Problem:** Not enough space for delivery

**Solutions:**
- Clear space before delivery
- Process items to floor immediately
- Request smaller, more frequent deliveries
- Optimize back room organization
- Communicate capacity limits to HQ

## Best Practices

### Receiving Efficiency Tips

1. **Prepare in Advance**
   - Clear receiving area
   - Have all equipment ready
   - Print expected items list

2. **Systematic Processing**
   - Open boxes in order
   - Scan items immediately
   - Keep track of progress

3. **Documentation Discipline**
   - Sign BOL with notes
   - Document issues immediately
   - File paperwork same day

4. **Team Communication**
   - Brief receiving team
   - Assign clear roles
   - Communicate discrepancies immediately
