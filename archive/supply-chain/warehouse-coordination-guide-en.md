## Purpose

This document provides comprehensive guidance on warehouse coordination for franchise stores. Effective coordination between stores and the distribution center ensures smooth supply chain operations, timely order fulfillment, and optimal inventory management.

### Key Objectives

| Objective | Impact |
|-----------|--------|
| **Clear Communication** | Reduce misunderstandings |
| **Efficient Operations** | Minimize delays |
| **Accurate Fulfillment** | Reduce errors |
| **Timely Resolution** | Quick issue handling |
| **Performance Tracking** | Continuous improvement |

## Examples / Scenarios

### Scenario 1: Standard Order Query

**Situation:** Store needs to check order status

**Process:**
```
1. Log into B2B Portal
2. Navigate to Order History
3. Find order number
4. Check status:
   - Pending → In processing queue
   - Processing → Being picked/packed
   - Shipped → In transit (click for tracking)
   - Delivered → Confirm receipt
5. If questions, submit ticket
6. Wait for response (24 hours)
```

---

### Scenario 2: Urgent Stock Availability Check

**Situation:** Customer wants item not in store

**Process:**
```
1. Check regional inventory in system
2. If unavailable locally, contact warehouse
3. Call warehouse hotline (urgent)
4. Provide:
   - SKU/item code
   - Quantity needed
   - Urgency level
5. Warehouse checks DC stock
6. If available:
   - Arrange expedited shipment
   - Confirm ETA with customer
7. If unavailable:
   - Check alternative options
   - Communicate to customer
```

---

### Scenario 3: Quality Issue Reporting

**Situation:** Received items with quality defects

**Process:**
```
1. Separate defective items
2. Photograph all defects
3. Document:
   - Order/delivery number
   - SKUs affected
   - Quantity with defects
   - Description of defects
4. Submit quality report via B2B Portal
5. Attach photos
6. Await warehouse response
7. Follow instructions for:
   - Return to DC
   - Local disposal
   - Credit/replacement
```

---

### Scenario 4: Delivery Schedule Change

**Situation:** Need to change delivery time

**Process:**
```
1. Check current delivery schedule
2. Contact warehouse at least 48 hours before
3. Request schedule change via:
   - B2B Portal (preferred)
   - Phone (urgent)
4. Provide:
   - Order number
   - Current delivery date
   - Requested new date/time
   - Reason for change
5. Receive confirmation
6. Update store schedule
```

## Common Issues & Solutions

### Issue 1: Order Delays

**Problem:** Orders taking longer than expected

**Solutions:**
- Check warehouse capacity/backlog
- Submit orders earlier
- Use priority shipping if urgent
- Communicate delays to customers
- Plan inventory buffer

---

### Issue 2: Stock Allocation Issues

**Problem:** Not receiving expected allocation

**Solutions:**
- Review allocation criteria
- Contact Regional Manager
- Submit allocation appeal with justification
- Consider alternative sources
- Request transfer from other stores

---

### Issue 3: Communication Breakdown

**Problem:** Not getting responses from warehouse

**Solutions:**
- Verify correct channel used
- Check ticket status in portal
- Follow up per escalation process
- Contact Regional Manager
- Document all attempts

---

### Issue 4: Recurring Quality Issues

**Problem:** Consistent product quality problems

**Solutions:**
- Document all issues systematically
- Compile trend report
- Submit formal quality complaint
- Request supplier investigation
- Implement incoming inspection

## Best Practices

### Effective Communication

1. **Be Clear and Specific**
   - Provide order/reference numbers
   - Include all relevant details
   - Attach supporting documentation

2. **Use Right Channel**
   - Standard: B2B Portal
   - Urgent: Phone
   - Complex: Email with follow-up

3. **Document Everything**
   - Save ticket numbers
   - Note conversation details
   - File all correspondence

4. **Follow Up Consistently**
   - Check status regularly
   - Escalate when needed
   - Confirm resolution
