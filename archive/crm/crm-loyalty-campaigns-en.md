## Purpose

This document defines the Loyalty Program and campaign management processes implemented at Aydınlı Group franchise stores (U.S. Polo Assn., Pierre Cardin, Cacharel).

### Loyalty Program Objectives

- Increase customer loyalty
- Improve repeat purchase rate
- Increase Customer Lifetime Value (CLV)
- Provide personalized experiences
- Strengthen customer data collection capabilities

## Points System

### Earning Points

| Condition | Points Rate | Example |
|-----------|-------------|---------|
| Standard Purchase | 1 Currency = 1 Point | 500 = 500 Points |
| Multiplier Campaign (2x) | 1 Currency = 2 Points | 500 = 1,000 Points |
| Multiplier Campaign (3x) | 1 Currency = 3 Points | 500 = 1,500 Points |
| Birthday Purchase | 1 Currency = 2 Points | During birthday week |

### Points Earning Rules

| Rule | Description |
|------|-------------|
| Matching required | Sale must be matched to customer to earn points |
| Discounted products | Points calculated on discounted sale amount |
| Return situation | Points from returned products are deducted |
| Validity | Points are valid for 12 months |

### Using Points

| Condition | Value | Rule |
|-----------|-------|------|
| Minimum Use | 100 Points | At least 100 points can be used |
| Point Value | 100 Points = 10 currency units | Every 100 points worth 10 |
| Maximum Use | 50% of cart value | At most half of cart value at once |
| Cash Refund | None | Points cannot be converted to cash |

### Points Validity

```
┌─────────────────────────────────────────────────────────────────┐
│                    POINTS VALIDITY CALENDAR                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Earned: January 2025        →    Expires: January 2026        │
│  Earned: June 2025           →    Expires: June 2026           │
│  Earned: December 2025       →    Expires: December 2026       │
│                                                                 │
│  ⚠️ Expired points are automatically deleted                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Coupon Management

### Coupon Properties

| Property | Description |
|----------|-------------|
| Coupon Code | Unique alphanumeric code |
| Discount Type | Percentage (%) or Amount |
| Minimum Cart | Minimum amount for coupon validity |
| Maximum Discount | Discount upper limit (for percentage coupons) |
| Validity | Start - End date |
| Usage Limit | Single use / Multiple use |
| Valid Products | All products / Specific categories |
| Excluded Products | Products outside campaign |

### Coupon Statuses

| Status | Description |
|--------|-------------|
| Active | Ready for use |
| Used | Already used |
| Expired | Validity date passed |
| Cancelled | Manually cancelled |
| Locked | Locked until certain condition met |

### Coupon Usage Rules

| Rule | Description |
|------|-------------|
| Single coupon | Only one coupon per sale |
| Points + Coupon | Both points and coupon can be used in same sale |
| Discounted product | Coupons may not be valid on discounted items |
| Return situation | Coupon not refunded on coupon sale return |

## Process Steps

### 1. Loyalty Membership Registration

**Step 1:** Introduce the program to customer

> "Our Aydınlı Club membership is completely free. You earn points on every purchase and benefit from exclusive campaigns."

**Step 2:** Create new customer record in CRM

**Step 3:** Obtain GDPR consents

**Step 4:** Check the loyalty program registration box

**Step 5:** If welcome bonus defined, inform the customer

**Step 6:** Provide membership information to customer

---

### 2. Earning Points During Sale

**Step 1:** Start the sale transaction

**Step 2:** Search for customer by phone number

**Step 3:** Match customer to sale

**Step 4:** Complete the sale

**Step 5:** System automatically calculates points

**Step 6:** Earned points appear on receipt

```
┌─────────────────────────────────────────┐
│           SALES RECEIPT                 │
├─────────────────────────────────────────┤
│  Item: Polo Shirt              350.00  │
│  Item: Linen Pants             650.00  │
│  ─────────────────────────────────────  │
│  TOTAL:                      1,000.00  │
│  ─────────────────────────────────────  │
│  Points Earned:                1,000   │
│  Total Points Balance:         3,500   │
└─────────────────────────────────────────┘
```

---

### 3. Points Redemption

**Step 1:** Match customer to sale

**Step 2:** Click "Use Points" option

**Step 3:** Enter points amount to use

| Option | Description |
|--------|-------------|
| Use All | Use all available points |
| Enter Amount | Use specific currency value of points |
| Enter Points | Enter specific points amount |

**Step 4:** Confirm discounted amount

**Step 5:** Take payment

```
┌─────────────────────────────────────────┐
│           SALES RECEIPT                 │
├─────────────────────────────────────────┤
│  TOTAL:                        500.00  │
│  Points Used: 2,000           -200.00  │
│  ─────────────────────────────────────  │
│  AMOUNT DUE:                   300.00  │
└─────────────────────────────────────────┘
```

---

### 4. Coupon Usage

**Step 1:** Match customer to sale

**Step 2:** Open "Coupons" tab

**Step 3:** View customer's active coupons

**Step 4:** Select appropriate coupon

**Step 5:** Check coupon conditions:
- [ ] Is minimum cart value met?
- [ ] Are products within coupon scope?
- [ ] Is coupon still valid?

**Step 6:** Click "Apply Coupon" button

**Step 7:** Verify discounted amount

**Step 8:** Take payment

---

### 5. Campaign Application

**Step 1:** Check current campaigns

**Step 2:** Evaluate customer's campaign eligibility

**Step 3:** Inform customer of applicable campaign

**Step 4:** Ensure cart conditions are met

**Step 5:** System automatically applies campaign

**Step 6:** Campaign discount appears on receipt

## KPIs

### Loyalty Program KPIs

| KPI | Definition | Formula | Target |
|-----|------------|---------|--------|
| **Loyalty Penetration** | Loyalty member sales rate | (Member Sales Count / Total Sales Count) × 100 | >70% |
| **Enrollment Rate** | New member registration rate | (New Members / New Customers) × 100 | >50% |
| **Active Member Rate** | Active member percentage | (Purchased in 12 Months / Total Members) × 100 | >40% |
| **Redemption Rate** | Points usage rate | (Points Used / Points Earned) × 100 | >30% |
| **Points Liability** | Unredeemed points value | Total Open Points × Point Value | Monitor |

### Campaign KPIs

| KPI | Definition | Formula | Target |
|-----|------------|---------|--------|
| **Campaign Response Rate** | Campaign participation rate | (Campaign Participants / Target Audience) × 100 | >15% |
| **Coupon Redemption Rate** | Coupon usage rate | (Coupons Used / Coupons Distributed) × 100 | >20% |
| **Incremental Sales** | Additional sales from campaign | Campaign Sales - Base Sales | Positive |
| **ROI** | Campaign return on investment | (Additional Revenue - Campaign Cost) / Campaign Cost | >2x |

### Calculation Examples

**Loyalty Penetration:**
```
Member sales count: 700
Total sales count: 1,000
Loyalty Penetration = (700 / 1,000) × 100 = 70%
```

**Redemption Rate:**
```
Points used this month: 500,000
Points earned this month: 1,500,000
Redemption Rate = (500,000 / 1,500,000) × 100 = 33%
```

## Common Issues & Solutions

### Issue 1: Coupon Not Working

**Checklist:**
- [ ] Has coupon expired?
- [ ] Is minimum cart value met?
- [ ] Are products within coupon scope?
- [ ] Has coupon been used before?
- [ ] Was customer matching done?

**Solution:** Check conditions, explain to customer if not met.

---

### Issue 2: Points Not Showing

**Possible Causes:**
- Sale not matched to customer
- System synchronization delayed
- Customer checking different profile

**Solution:**
1. Check matching from sales history
2. Wait 15-30 minutes
3. If problem persists, open ticket

---

### Issue 3: Customer Used Wrong Points Amount

**Solution:**
1. Cancel transaction (store manager approval required)
2. Create new transaction
3. Use correct points amount
4. Document situation with ticket

---

### Issue 4: Campaign Not Applying

**Checklist:**
- [ ] Are campaign dates valid?
- [ ] Is customer in target segment?
- [ ] Are cart conditions met?
- [ ] Are products within campaign scope?

**Solution:** Check campaign conditions from CRM.

## Store Tips

### Increasing Loyalty Sales

1. **Ask every customer:** "Are you an Aydınlı Club member?"
2. **Explain benefits:** "You can earn 500 points right now"
3. **Remind points:** "You have 3,500 points, would you like to use them today?"
4. **Announce campaigns:** "There's an exclusive 20% discount for members"

### Encouraging Membership Registration

| Approach | Example |
|----------|---------|
| Instant benefit | "Sign up now, get a 50 welcome coupon" |
| Future benefit | "Earn points to use on your next purchase" |
| Special occasion | "Let us send you a special discount on your birthday" |
| Early access | "See new collections before everyone else" |

### Campaign Application Rules

| Do | Don't |
|----|-------|
| ✅ Explain campaign conditions | ❌ Don't hide conditions |
| ✅ Suggest the most advantageous option | ❌ Don't mislead customer |
| ✅ Check coupon/points combination | ❌ Don't apply off-system discounts |
| ✅ Inform earned benefits after transaction | ❌ Don't accept invalid coupons |

### Difficult Situations

| Situation | Approach |
|-----------|----------|
| Customer missed campaign | Inform about upcoming campaign |
| Points expired | Respond with understanding, encourage new earning |
| Coupon not working | Patiently explain conditions |
| System error | Apologize, open ticket, inform customer for follow-up |
