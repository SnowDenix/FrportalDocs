## Daily Operation Procedures

### Daily Operation Flow

```
08:30 - Store Opening Preparation
09:00 - Store Opening
09:00-12:00 - Morning Shift
12:00-13:00 - Lunch Break (Rotation)
13:00-18:00 - Afternoon Shift
18:00-21:00 - Evening Shift
21:00-21:30 - Store Closing
```

### Shift Planning

| Shift | Time | Minimum Staff |
|-------|------|---------------|
| Morning | 08:30-17:00 | 2 people |
| Evening | 13:00-21:30 | 2 people |
| Full Day | 09:00-21:30 | (Rotation) |

### Staff Role Distribution

| Position | Responsibilities |
|----------|------------------|
| **Store Manager** | General management, reporting, staff |
| **Shift Supervisor** | Shift management, cash control |
| **Sales Associate** | Customer service, sales, tidying |
| **Cashier** | Payment processing, returns |
| **Stock Room Associate** | Goods receipt, inventory management |

## Store Opening Procedure

### Opening Checklist (30 mins before opening)

#### 1. Security Checks
- [ ] Disable alarm system
- [ ] Check all entry/exit points
- [ ] Any overnight abnormalities?
- [ ] Check security cameras

#### 2. Physical Preparation
- [ ] Turn on all lights
- [ ] Adjust AC/heating
- [ ] Check floor cleanliness
- [ ] Check windows

#### 3. Store Layout
- [ ] Window display check
- [ ] Fixture arrangement check
- [ ] Mannequin check
- [ ] Signage check
- [ ] Price tag check

#### 4. Cash Register Preparation
- [ ] Opening cash count
- [ ] Turn on and test POS devices
- [ ] Sufficient change available?
- [ ] Receipt roll check

#### 5. Stock Preparation
- [ ] Any overnight deliveries?
- [ ] Replenish missing sizes/colors
- [ ] Check products for return

## Store Closing Procedure

### Closing Checklist (Start 30 mins before closing)

#### 1. Customer Management
- [ ] Inform last customer (15 mins before closing)
- [ ] Monitor customers in fitting rooms
- [ ] Make closing announcement (if applicable)

#### 2. Cash Register Closing
- [ ] Print Z report
- [ ] Cash count
- [ ] Cash vs credit card reconciliation
- [ ] Lock cash drawer
- [ ] Prepare daily report

#### 3. Store Layout
- [ ] Place all products
- [ ] Empty and tidy fitting rooms
- [ ] Collect hangers
- [ ] Floor cleaning
- [ ] Window check

#### 4. Security
- [ ] Check back rooms
- [ ] Lock stockroom door
- [ ] Turn off all lights (except security lights)
- [ ] Turn off AC
- [ ] Activate alarm system
- [ ] Close shutters/gates

## Return and Exchange Procedures

### Return Policy

| Condition | Period | Requirement | Action |
|-----------|--------|-------------|--------|
| Tagged, unused | 14 days | With receipt | Full refund |
| Tagged, unused | 14-30 days | With receipt | Store credit |
| Untagged/used | - | - | No return accepted |
| Defective product | 30 days | With receipt | Exchange or refund |

### Return Process Steps

```
1. Greet Customer
   ↓
2. Receipt/Invoice Check
   ↓
3. Product Check (tag, condition)
   ↓
4. Determine Return Reason
   ↓
5. System Entry (POS)
   ↓
6. Complete Return Form
   ↓
7. Refund / Exchange
   ↓
8. Customer Signature
```

### Return Reason Codes

| Code | Reason | Description |
|------|--------|-------------|
| R01 | Size mismatch | Customer bought wrong size |
| R02 | Color dissatisfaction | Customer doesn't like color |
| R03 | Gift return | Gift recipient returning |
| R04 | Manufacturing defect | Product is defective |
| R05 | Price difference | Found cheaper elsewhere |
| R06 | Other | Explanation required |

## Damage and Rejection Procedures

### Damage Types

| Type | Definition | Action |
|------|------------|--------|
| **Manufacturing Defect** | Factory origin | Rejection report, return to HQ |
| **Transit Damage** | During shipment | Report, insurance claim |
| **Store Damage** | Occurred in store | Shrinkage record |
| **Customer Damage** | Customer caused | Shrinkage or compensation |

### Damage Detection Procedure

```
1. Detect Damage
   ↓
2. Take Photos (min. 3 angles)
   ↓
3. Complete Damage Form
   ↓
4. Separate Product (red tag)
   ↓
5. Notify HQ (within 24 hours)
   ↓
6. Await Approval
   ↓
7. Process Shrinkage/Rejection
```

### Rejection Procedure

In case of delivery rejection:

1. Inspect upon delivery
2. Identify damaged/wrong items
3. Note on delivery document
4. Get carrier signature
5. Prepare rejection report
6. Notify HQ within 24 hours
7. Store product separately

## Price Change Procedures

### Price Change Types

| Type | Definition | Authority |
|------|------------|-----------|
| **Central Discount** | General campaign | Headquarters |
| **Regional Discount** | Region-based | Regional Manager |
| **Markdown** | End of season | HQ approval |
| **Price Correction** | Error correction | Store Manager |

### Price Update Procedure

```
1. HQ Price Change Notification
   ↓
2. Download Price List
   ↓
3. Update System (POS)
   ↓
4. Physical Tag Change
   ↓
5. Verification (random 10 products)
   ↓
6. Completion Notification
```

### Tag Change Rules

- Old tags must be completely removed
- No overlapping tags
- No handwriting on tags
- All sizes must be updated

1. Receive transfer request
2. Collect products
3. Count and verify
4. System exit (GI)

## Cash Management

### Payment Types

| Payment | Process | Points of Attention |
|---------|---------|-------------------|
| Cash | Count, change | Counterfeit check |
| Credit Card | POS transaction | Signature/PIN verification |
| Debit Card | POS transaction | Balance check |
| Gift Card | System entry | Validity check |
| Installment | POS installment | Installment options |

### Cash Shortage/Overage Procedure

**In Case of Shortage:**
1. Recount
2. Check day's transactions
3. Review camera footage
4. Prepare report
5. Notify store manager

**In Case of Overage:**
1. Recount
2. Check return/void transactions
3. Customer change check
4. Prepare report

### Daily Cash Limits

| Amount | Action |
|--------|--------|
| < $1,000 | Normal operation |
| $1,000-$2,500 | Intra-shift bank deposit |
| > $2,500 | Urgent bank transfer |

## Customer Service Standards

### Customer Greeting

```
1. Make eye contact
2. Smile
3. Say "Welcome"
4. Approach within 3 minutes
5. Ask "How may I help you?"
```

### Sales Process

```
Needs Analysis → Product Presentation → Try-on → Objection Handling → Closing
```

### Complaint Management

| Step | Action |
|------|--------|
| 1 | Listen (don't interrupt) |
| 2 | Show empathy |
| 3 | Apologize |
| 4 | Offer solution |
| 5 | Follow up |

### Prohibited Behaviors

- Arguing with customers
- Long personal phone calls
- Eating/drinking (on sales floor)
- Personal conversations (when customers present)
- Negative body language

## Emergency Procedures

### Fire

1. Activate fire alarm
2. Call emergency services
3. Evacuate customers
4. Use fire extinguisher (small fire)
5. Use emergency exits
6. Go to assembly point

### Theft

1. Notify security
2. Don't intervene (if dangerous)
3. Identify the person
4. Secure camera footage
5. Prepare report
6. Call police (if necessary)

### Medical Emergency

1. Call emergency services
2. Apply first aid (if trained)
3. Don't move the person
4. Stay calm
5. Clear the area
