## Giriş ve Amaç

Visual Merchandising (VM), ürünlerin mağaza içinde görsel olarak en etkili şekilde sergilenmesini sağlayan stratejik bir disiplindir.

### VM'nin Hedefleri

- **Satış Artışı:** Doğru ürün yerleşimi ile impuls satışları artırmak
- **Marka Algısı:** Tutarlı marka imajı oluşturmak
- **Müşteri Deneyimi:** Alışveriş sürecini keyifli hale getirmek
- **Stok Verimliliği:** Ürünlerin dengeli şekilde satışını sağlamak

### VM'nin Önemi

```
Etkili VM = Daha Fazla Ziyaretçi × Yüksek Dönüşüm × Yüksek UPT × Yüksek ATV
```

## Temel VM Prensipleri

### 1. Altın Üçgen Kuralı

Müşteri giriş noktasından bakıldığında, en önemli ürünler "Altın Üçgen" bölgesinde sergilenmelidir.

```
        [VİTRİN]
           △
          / \
         /   \
        /     \
       /  ★★★  \    ← Altın Üçgen Bölgesi
      /         \
     /           \
    /_____________\
       [GİRİŞ]
```

### 2. Göz Seviyesi Kuralı

- **Premium Zone (145-170 cm):** En karlı, yeni sezon ürünleri
- **Touch Zone (90-145 cm):** Kolay erişim, deneme ürünleri
- **Lower Zone (0-90 cm):** Hacimli, ağır ürünler
- **Upper Zone (170+ cm):** Stok, görsel dekor

### 3. Sağa Dönüş Prensibi

Müşterilerin %90'ı mağazaya girince sağa döner. Bu nedenle:
- Sağ taraf: Yeni sezon, trend ürünler
- Sol taraf: Temel ürünler, outlet

### 4. Dekompresson Zone

Giriş alanının ilk 1-2 metresi müşterinin mağazaya adapte olduğu bölgedir. Bu alana kritik ürün yerleştirmeyin.

## Mağaza Bölgeleri ve Düzeni

### A-B-C Zone Sistemi

| Zone | Konum | Ürün Tipi | Satış Potansiyeli |
|------|-------|-----------|-------------------|
| **A Zone** | Giriş, vitrin önü | Hero items, yeni sezon | En yüksek |
| **B Zone** | Orta alan | Trend, sezonsal | Orta-yüksek |
| **C Zone** | Arka bölge | Temel, basics | Düşük-orta |

### Alan Dağılımı (Örnek 100 m² mağaza)

```
┌─────────────────────────────────────────┐
│              VİTRİN                      │
├─────────────────────────────────────────┤
│  A ZONE   │      B ZONE      │  A ZONE  │
│  (Erkek)  │    (Aksesuar)    │ (Kadın)  │
│   25 m²   │      15 m²       │   25 m²  │
├───────────┼──────────────────┼──────────┤
│           │    C ZONE        │          │
│   KASA    │   (Basics)       │  KABİN   │
│           │     20 m²        │          │
├───────────┴──────────────────┴──────────┤
│            DEPO (15 m²)                  │
└─────────────────────────────────────────┘
```

### Trafik Akışı Planlaması

- **Ana Yollar:** Min. 120 cm genişlik
- **Ara Yollar:** Min. 90 cm genişlik
- **Fixture Arası:** Min. 80 cm

## Vitrin Standartları

### Vitrin Değişim Takvimi

| Dönem | Değişim Sıklığı |
|-------|-----------------|
| Yeni Sezon Lansmanı | Anında |
| Kampanya Dönemleri | Kampanya başı |
| Normal Dönem | 2 haftada bir |
| Özel Günler | Etkinlik öncesi |

### Vitrin Kompozisyon Kuralları

#### Piramit Düzeni
```
       ▲
      /█\
     / █ \
    / █ █ \
   / █ █ █ \
  ───────────
```
- Orta nokta en yüksek
- Simetrik veya asimetrik olabilir
- Bakış noktası 145-165 cm

#### Repetition (Tekrar) Düzeni
```
  █   █   █   █   █
  ▼   ▼   ▼   ▼   ▼
```
- Aynı ürünün farklı renkleri
- Güçlü görsel etki

### Vitrin Aydınlatma

- **Genel Aydınlatma:** 800-1000 lux
- **Spot Aydınlatma:** 2000-3000 lux (hero ürünlere)
- **Renk Sıcaklığı:** 3000-4000K (sıcak beyaz)

### Vitrin Manken Kullanımı

| Mağaza Büyüklüğü | Manken Sayısı |
|------------------|---------------|
| < 80 m² | 2-3 |
| 80-150 m² | 4-6 |
| > 150 m² | 6-8 |

## Fixture ve Ürün Yerleşimi

### Fixture Türleri

#### 1. Wall Fixtures (Duvar Üniteleri)
- **Mid-Wall:** 150 cm yükseklik, ana ürünler
- **High-Wall:** 220 cm yükseklik, görseller + stok

#### 2. Floor Fixtures (Zemin Üniteleri)
- **Gondol:** Çift taraflı, orta alan
- **4-Way:** Dört yönlü, aksesuar
- **Round Rail:** Dairesel askılık
- **Nesting Table:** İç içe masalar

### Ürün Sunumu Standartları

#### Katlama Standartları

| Ürün Tipi | Katlama Şekli | Yığın Yüksekliği |
|-----------|---------------|------------------|
| T-Shirt | Yarım katlama | Max 8 adet |
| Polo | Yakalı katlama | Max 6 adet |
| Gömlek | Tam katlama + klips | Max 5 adet |
| Pantolon | Yarım katlama | Max 6 adet |

#### Askı Yönlendirmesi

```
Müşteri Bakış Yönü →

   ←───────────────
   Askı açıklığı sola bakmalı
```

#### Beden Sıralaması

- Soldan sağa: XS → S → M → L → XL → XXL
- Yukarıdan aşağıya: Küçük → Büyük

### Kapasite Standartları

| Fixture Tipi | Max Ürün Sayısı |

## Renk ve Tema Yönetimi

### Renk Blokları Oluşturma

Ürünleri renk geçişlerine göre gruplandırın:

```
AÇIK ──────────────────────────────────► KOYU

Beyaz → Krem → Sarı → Turuncu → Kırmızı → Bordo → Lacivert → Siyah
```

### Renk Hikayesi Anlatımı

Her fixture'da maksimum 3-4 renk kombinasyonu kullanın:

**Doğru:**
```
[Lacivert] [Beyaz] [Kırmızı]
```

**Yanlış:**
```
[Yeşil] [Sarı] [Mor] [Turuncu] [Pembe] [Mavi]
```

### Sezonsal Renk Paleti

| Sezon | Ana Renkler | Aksan Renkler |
|-------|-------------|---------------|
| SS (İlkbahar/Yaz) | Beyaz, Mavi, Mint | Coral, Sarı |
| FW (Sonbahar/Kış) | Lacivert, Bordo, Kahve | Hardal, Bakır |

## VM Kontrol Listesi

- [ ] Tüm ürünler düzgün asılı/katlanmış mı?
- [ ] Beden sıralaması doğru mu?
- [ ] Boş askı/raf var mı?
- [ ] Fiyat etiketleri görünür mü?
- [ ] Aydınlatma çalışıyor mu?
- [ ] Zemin temiz mi?
- [ ] Mankenler düzgün mü?
