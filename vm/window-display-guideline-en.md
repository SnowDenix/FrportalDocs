## Purpose

This document establishes comprehensive guidelines for window display design, installation, and maintenance across all Aydınlı Group franchise stores. Window displays serve as the primary brand touchpoint and the most powerful traffic-driving tool in retail.

### Window Display Objectives

| Objective | Impact Metric |
|-----------|---------------|
| **Attract Attention** | Increase foot traffic by 15-25% |
| **Communicate Brand Story** | Brand recall improvement |
| **Showcase Key Products** | Hero product sell-through |
| **Drive Store Entry** | Conversion from passersby |
| **Support Campaigns** | Campaign awareness |

### The 3-Second Rule

Customers make their decision to enter a store within **3 seconds** of seeing the window. Every element must communicate instantly:

```
┌─────────────────────────────────────────────────────┐
│              THE 3-SECOND DECISION                  │
├─────────────────────────────────────────────────────┤
│                                                     │
│   Second 1: NOTICE                                  │
│   └── Visual impact captures attention              │
│                                                     │
│   Second 2: UNDERSTAND                              │
│   └── Message/offer comprehended                    │
│                                                     │
│   Second 3: DECIDE                                  │
│   └── Customer chooses to enter or walk away        │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Mannequin Dressing Procedure

```
Step 1: PREPARATION
├── Select correct outfit per styling guide
├── Steam all garments to remove wrinkles
├── Verify all buttons, zippers functional
├── Prepare pins, tapes, and clips
└── Gather all accessories

Step 2: BASE LAYER
├── Start with trousers/bottoms
├── Ensure correct fit and length
├── Pin excess fabric at back
└── Adjust break at shoe

Step 3: TOP LAYER
├── Add shirt/top layer
├── Button correctly per brand standard
├── Tuck or leave untucked per guide
└── Adjust collar and cuffs

Step 4: OUTERWEAR (if applicable)
├── Add jacket/coat
├── Position lapels correctly
├── Adjust sleeves to show cuffs
└── Button per styling guide

Step 5: ACCESSORIES
├── Add belt if applicable
├── Position tie/scarf
├── Add watch/jewelry
├── Place bag/accessories
└── Add shoes last

Step 6: FINAL ADJUSTMENT
├── Step back and view from distance
├── Adjust pose if articulated mannequin
├── Ensure natural drape
├── Pin any loose areas
└── Final quality check
```

#### Pyramid Composition
```
                    ▲
                   /█\
                  / █ \
                 / █ █ \
                /█  █  █\
               ───────────

Height at center, descending to sides
```

```
        █
        █   ▲
        █  /█\
        █ /███\
       ─────────

• Modern, fashion-forward
• Use for trend statements

```

```
   ▼   ▼   ▼   ▼   ▼
```

### Lighting Standards

| Lighting Type | Purpose | Specification |
|---------------|---------|---------------|
| **General Wash** | Overall illumination | 800-1000 lux |
| **Accent Spotlights** | Hero product highlight | 2000-3000 lux |
| **Color Temperature** | Brand ambiance | 3000-4000K (warm white) |
| **Backlight** | Depth and drama | 500-800 lux |

### Lighting Angle Guide

```
Side View - Spotlight Positioning

         Ceiling
    ═══════════════════════════════
              \
               \  30-45°
                \  angle
                 \
                  ○ ← Mannequin
                 ───
                Floor

• Primary spots at 30-45° from vertical
• Avoid flat front lighting
• Create shadows for dimension
• Multiple sources for natural effect
```

## Checklists

### Pre-Change Checklist

```
INVENTORY CHECK
□ Window kit received complete
□ All products available in correct sizes
□ Props and signage undamaged
□ Mannequin parts/wigs available
□ Cleaning supplies ready
□ Tools available (pins, tape, clips, etc.)

TEAM PREPARATION
□ Team members assigned
□ Timing confirmed (preferably after hours)
□ Installation guide reviewed by all
□ Reference photos printed/accessible
□ Contact for questions available

TECHNICAL CHECK
□ All spotlights functional
□ Spare bulbs available
□ Electrical connections working
□ Steamer operational
□ Step ladder/platform safe
```

### Installation Quality Checklist

```
COMPOSITION
□ Matches reference photo layout
□ Focal point clearly established
□ Pyramid/asymmetric balance achieved
□ Mannequin spacing appropriate
□ Props positioned correctly

MANNEQUIN STYLING
□ Outfits match styling guide exactly
□ All garments pressed/wrinkle-free
□ Fit adjusted with pins/tape as needed
□ Accessories complete and correct
□ Shoes clean and positioned

SIGNAGE & GRAPHICS
□ Signage straight and secure
□ Prices/offers current and accurate
□ Brand logo visible and correct
□ No damaged or faded graphics
□ Campaign message clear

LIGHTING
□ All lights functional
□ Hero products properly highlighted
□ No harsh shadows on faces
□ Even coverage across display
□ Dramatic effect achieved

CLEANLINESS
□ Glass spotless inside and out
□ Floor/platform clean
□ No fingerprints on surfaces
□ Mannequins dust-free
□ No visible tape or pins from front
```

### Daily Window Maintenance Checklist

```
MORNING CHECK (Before Opening)
□ Clean interior window glass
□ Verify all lighting working
□ Check mannequin positioning unchanged
□ Confirm products still available in store
□ Remove any overnight dust/debris

AFTERNOON CHECK (Mid-Day)
□ Quick visual inspection from outside
□ Adjust any shifted elements
□ Replace any sold products if needed
□ Verify lighting still optimal

EVENING CHECK (Before Closing)
□ Final visual inspection
□ Clean fingerprints from glass
□ Straighten any moved items
□ Report any issues for next day
```

## Common Issues & Solutions

### Issue 1: Window Kit Arrives Incomplete

**Problem:** Missing components in window kit delivery

**Immediate Action:**
1. Document missing items with photos
2. Contact HQ VM team immediately
3. Check if items can be sourced locally
4. Implement partial installation if possible

**Solution:**
- Use previous season backup props if appropriate
- Substitute similar available products
- Request expedited shipping of missing items

**Prevention:** Detailed kit inventory upon receipt, 72 hours before change

---

### Issue 2: Mannequin Damage

**Problem:** Broken mannequin parts (hand, arm, base)

**Immediate Action:**
1. Remove damaged mannequin from display
2. Rearrange remaining mannequins to cover gap
3. Report damage immediately

**Solution:**
- Use spare parts from inventory
- Order replacement parts
- Temporarily reduce mannequin count while maintaining composition

**Prevention:** Careful handling training, proper storage, regular inspection

---

### Issue 3: Lighting Failure

**Problem:** Spotlights not working or uneven lighting

**Immediate Action:**
1. Check bulb - replace if burnt
2. Check electrical connection
3. Adjust remaining lights to compensate

**Solution:**
- Replace bulbs (keep spare inventory)
- Call electrician for complex issues
- Use temporary lighting if needed

**Prevention:** Weekly lighting check, maintain spare bulb inventory

---

### Issue 4: Window Doesn't Match Reference

**Problem:** Installed window differs significantly from HQ guide

**Immediate Action:**
1. Take photos and compare point-by-point
2. Identify specific discrepancies
3. Make immediate corrections where possible

**Solution:**
- Re-read installation guide carefully
- Request video call with VM team for guidance
- Document correct process for future

**Prevention:** Step-by-step installation following guide, reference photos visible during installation

---

### Issue 5: Products Sell Out During Window Cycle

**Problem:** Display products no longer available in store

**Immediate Action:**
1. Check if products available in back stock
2. Contact nearby stores for transfer
3. Identify similar alternative products

**Solution:**
- Replace with same style, different color
- Update with next best-seller
- Adjust composition if necessary

**Prevention:** Higher stock allocation for window products, daily availability check

---

### Issue 6: Weather/Environmental Damage

**Problem:** Sun fading, condensation, or environmental impact

**Immediate Action:**
1. Rotate products to minimize sun exposure
2. Address condensation sources
3. Document damage for claims if needed

**Solution:**
- Apply UV film to window if chronic issue
- Improve ventilation
- More frequent product rotation

**Prevention:** UV protection, climate control, product rotation schedule
