## Purpose

This document establishes comprehensive guidelines for mannequin selection, styling, maintenance, and presentation across all Aydınlı Group franchise stores. Mannequins serve as silent salespeople and are critical to communicating brand identity and driving product sales.

### Mannequin Impact on Sales

| Metric | Impact |
|--------|--------|
| **Outfit Sell-Through** | Mannequin outfits sell 2-3x faster |
| **UPT Increase** | Complete looks drive +25% UPT |
| **Brand Perception** | Premium styling = premium perception |
| **Customer Inspiration** | 65% of customers seek style guidance |

## Process Steps

### Mannequin Dressing Procedure

```
┌────────────────────────────────────────────────────────────────┐
│                 MANNEQUIN DRESSING PROCEDURE                   │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   STEP 1: PREPARATION                                          │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ □ Review look book for approved outfit                  │  │
│   │ □ Gather all garments in correct sizes                  │  │
│   │ □ Steam all items to remove wrinkles                    │  │
│   │ □ Check for missing buttons, loose threads              │  │
│   │ □ Prepare accessories (belt, tie, watch, etc.)          │  │
│   │ □ Gather tools: pins, tape, clips, tissue paper         │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   STEP 2: UNDRESSING (If re-dressing)                          │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ □ Remove accessories first (jewelry, watch, belt)       │  │
│   │ □ Remove outerwear (jacket, coat)                       │  │
│   │ □ Remove top layer (shirt, sweater)                     │  │
│   │ □ Remove bottom layer (trousers)                        │  │
│   │ □ Remove shoes last                                     │  │
│   │ □ Collect all pins and clips                            │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   STEP 3: BASE LAYER                                           │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ □ Start with trousers/pants                             │  │
│   │ □ Pull up to natural waist position                     │  │
│   │ □ Check correct length at hem                           │  │
│   │ □ Pin excess fabric at back (invisible from front)      │  │
│   │ □ Create proper break at shoe                           │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   STEP 4: MID LAYER                                            │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ □ Add shirt/polo/t-shirt                                │  │
│   │ □ Button correctly per brand standard                   │  │
│   │ □ Tuck in or leave out per look book                    │  │
│   │ □ Adjust collar position                                │  │
│   │ □ Roll sleeves if indicated                             │  │
│   │ □ Pin excess fabric at back                             │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   STEP 5: OUTER LAYER (If applicable)                          │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ □ Add blazer/jacket/coat                                │  │
│   │ □ Position shoulders correctly                          │  │
│   │ □ Adjust lapels symmetrically                           │  │
│   │ □ Button per style guide (usually middle button)        │  │
│   │ □ Show shirt cuffs (1-2 cm)                             │  │
│   │ □ Adjust back vents                                     │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   STEP 6: ACCESSORIES                                          │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ □ Add belt (matching shoe color)                        │  │
│   │ □ Add tie/scarf if applicable                           │  │
│   │ □ Add watch (always on left wrist)                      │  │
│   │ □ Add bag/briefcase if indicated                        │  │
│   │ □ Position glasses/sunglasses                           │  │
│   │ □ Add shoes last (clean and polished)                   │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
│   STEP 7: FINAL ADJUSTMENTS                                    │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │ □ Step back and view from customer angle                │  │
│   │ □ Check overall silhouette                              │  │
│   │ □ Ensure natural drape of fabrics                       │  │
│   │ □ Hide all pins from visible angles                     │  │
│   │ □ Adjust pose if articulated mannequin                  │  │
│   │ □ Final steam touch-up if needed                        │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Universal Styling Rules

#### DO's ✓

| Rule | Reason |
|------|--------|
| Steam all garments before dressing | Wrinkles look unprofessional |
| Use correct mannequin size | Proper fit is essential |
| Pin excess fabric at back | Creates fitted appearance |
| Match belt to shoes | Shows attention to detail |
| Show 1-2 cm of shirt cuff | Proper proportions |
| Keep tags hidden | Customer-ready appearance |
| Use tissue in shoulders | Maintains jacket shape |

#### DON'Ts ✗

| Rule | Reason |
|------|--------|
| Use damaged mannequins | Damages brand perception |
| Leave hangers visible | Looks unfinished |
| Mix brands on one mannequin | Brand integrity |
| Over-accessorize | Distracts from products |
| Use clashing colors | Confuses customer |
| Ignore season appropriateness | Relevance to customer needs |
| Leave visible pins from front | Unprofessional appearance |

## Checklists

### Daily Mannequin Check

```
WINDOW MANNEQUINS
□ All outfits complete (no missing items)
□ No visible pins or tape from outside
□ All buttons fastened correctly
□ Accessories in place
□ Shoes clean and positioned
□ No damage to mannequin surface
□ Products still in stock

IN-STORE MANNEQUINS
□ Styling maintained from opening
□ Products available for sale
□ No fallen accessories
□ Poses maintained
□ Cleanliness of mannequin surface
```

### Styling Quality Checklist

```
GARMENT QUALITY
□ All items steamed/pressed
□ No missing buttons
□ No loose threads
□ Correct size for mannequin
□ Tags hidden
□ Garment damage-free

FIT & PRESENTATION
□ Natural drape achieved
□ No visible excess fabric from front
□ Correct trouser break
□ Collar positioned correctly
□ Sleeves at proper length
□ Jacket sits on shoulders properly

ACCESSORIES
□ Belt matches shoes
□ Watch on left wrist
□ Tie properly knotted (if applicable)
□ Pocket square folded correctly
□ All accessories clean
□ Appropriate for outfit style

OVERALL IMPRESSION
□ Outfit tells a story
□ Brand identity clear
□ Season appropriate
□ Customer can visualize wearing
□ Complete look achievable in store
```

## Common Issues & Solutions

### Issue 1: Poor Garment Fit

**Problem:** Clothes look too big/small on mannequin

**Solution:**
- Select correct mannequin size for garment
- Use strategic pinning at back
- Add tissue paper for structure
- Choose appropriate garment size

**Prevention:** Maintain size guide for each mannequin

---

### Issue 2: Wrinkled Garments

**Problem:** Visible wrinkles despite steaming

**Solution:**
- Re-steam immediately before dressing
- Use proper steaming technique
- Store garments properly
- Check steamer water level

**Prevention:** Steam just before installation

---

### Issue 3: Products Constantly Selling Out

**Problem:** Mannequin items unavailable for customers

**Solution:**
- Increase stock allocation for mannequin items
- Create "mannequin product" flag in inventory
- Establish quick replacement protocol
- Use slightly different colorway as backup

**Prevention:** Reserve stock for mannequin looks

---

### Issue 4: Mannequin Damage

**Problem:** Chips, scratches, or broken parts

**Solution:**
- Remove from display immediately
- Use touch-up paint for minor chips
- Order replacement parts
- Rotate with backup mannequins

**Prevention:** Proper handling training, careful storage

---

### Issue 5: Inconsistent Styling Across Stores

**Problem:** Same look book, different execution

**Solution:**
- Share reference photos
- Conduct video training sessions
- Create detailed step-by-step guides
- Regular audit and feedback

**Prevention:** Standardized training program
